The result of an academic-industrial collaboration, Vibrosoft was specifically developed for the GAPS group (Signal Analysis and Processing Group) at the Technological University of Havana CUJAE. It unravels the complexities of interaction between devices and data acquisition systems. It implements a user management system with granular roles and permissions, using advanced authentication techniques. The software demonstrates the ability to handle real-time communications, implement fault detection systems, and create an interface that allows secure remote interaction with monitoring devices.
